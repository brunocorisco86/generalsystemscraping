# Catálogos do Sistema

## Tipos de Exploração

Os seguintes tipos de exploração são suportados pelo sistema e devem ser utilizados para classificar as estruturas:

| ID | Nome | Descrição |
|----|------|-----------|
| 1 | Piscicultura | Criação de peixes em tanques ou açudes. |
| 2 | Avicultura | Criação de aves (frangos, poedeiras). |
| 3 | Suinocultura | Criação de suínos. |
| 4 | Bovinocultura de Leite | Produção leiteira. |
| 5 | Bovinocultura de Corte | Criação de gado para corte. |
| 6 | Caprinocultura | Criação de cabras. |
| 7 | Ovinocultura | Criação de ovelhas. |

## Unidades de Medida Sugeridas

- **Oxigênio**: mg/L
- **Temperatura**: °C
- **Amônia/Nitrito**: mg/L
- **Alcalinidade**: mg/L CaCO3
- **Transparência**: cm (Disco de Secchi)
- **SDT (Sólidos Dissolvidos Totais)**: ppm
- **ORP (Potencial de Oxirredução)**: mV
- **Cloro**: ppm
